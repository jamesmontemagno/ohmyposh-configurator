üí° What is MCP?

The Model Context Protocol (MCP) is an open standard that enables AI assistants to interact with external tools and data sources.

Our MCP server allows AI assistants like Claude, GitHub Copilot, and others to help you create and manage Oh My Posh configurations through natural language conversations.

Example:

"Create me a developer prompt with git and Python info"

‚Üí AI generates a complete Oh My Posh configuration with appropriate segments, colors, and templates!

‚öôÔ∏è Setup Instructions

Get started in minutes with your preferred AI assistant:

Clone and Build

First, clone the repository and build the MCP server:

git clone https://github.com/jamesmontemagno/ohmyposh-configurator.git
cd ohmyposh-configurator
npm install
npm run build:mcp

Configure Your AI Assistant

For Claude Desktop:

Add to your claude_desktop_config.json file:

macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
Windows: %APPDATA%\Claude\claude_desktop_config.json 

{
  "mcpServers": {
    "ohmyposh": {
      "command": "node",
      "args": [
        "/absolute/path/to/ohmyposh-configurator/dist/mcp/index.js"
      ]
    }
  }
}

For VS Code with GitHub Copilot:

Install an MCP-compatible extension and configure it to use:

/absolute/path/to/ohmyposh-configurator/dist/mcp/index.js

Start Creating!

Restart your AI assistant and start creating configurations with natural language.

Try: "Create a developer prompt with git and Python"
